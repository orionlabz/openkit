# API

## Surface

| Type | Name | Location | Notes |
|---|---|---|---|
| CLI | `openkit` root command | `rust-cli/src/main.rs` | Main command entrypoint. |
| CLI | `openkit --version` | `rust-cli/src/main.rs` | Prints runtime version. |
| CLI | `openkit upgrade` | `rust-cli/src/main.rs` | Rust-native self-update on Linux/macOS (checksum + atomic swap); installer flow on Windows. |
| CLI | `openkit upgrade --check` | `rust-cli/src/main.rs` | Checks latest GitHub release tag. |
| CLI | `openkit uninstall` | `rust-cli/src/main.rs` | Removes installed OpenKit binary from known install paths with confirmation. |
| CLI | `openkit check` | `rust-cli/src/main.rs` | Agent/system availability checks. |
| CLI | `openkit check --json` | `rust-cli/src/main.rs` | Machine-readable health output. |
| CLI | `openkit init` | `rust-cli/src/main.rs` | Initialize baseline project docs and structure. |
| CLI | `openkit <agent> sync` | `rust-cli/src/main.rs` | Baseline agent config sync (`.agent/OPENKIT.md`). |
| CLI | `openkit <agent> doctor` | `rust-cli/src/main.rs` | Baseline agent config health report. |
| CLI | `openkit <agent> upgrade` | `rust-cli/src/main.rs` | Baseline sync with overwrite semantics. |
| CLI | `openkit memory init|doctor|capture|review` | `rust-cli/src/main.rs` | Docs-first Memory Kernel runtime commands. |

## Contracts

- CLI behavior is implemented with `clap` command parsing.
- Memory command contracts are validated by `rust-cli/tests/command_contracts.rs`.
- Release artifacts are generated by `.github/workflows/release.yml`.

## Gaps

- Agent sync/upgrade/doctor baseline parity is available; advanced parity remains in progress (Sprint-09+).
- No HTTP/REST/GraphQL API surface found.
