# OpenKit CLI Documentation

This folder is the source of truth for OpenKit CLI planning and documentation.

## Sections

- `content-protocol/`: canonical pack format, schemas, and sync semantics
- `agent-compat/`: per-agent layout notes and constraints
- `requirements/`: feature specs and implementation plans (SDD gate)
- `sprint/`: sprint goals, backlog, tasks, and risk register
- `adr/`: architecture decision records for the CLI product
