# Memory Kernel Templates

This directory contains docs-first Memory Kernel templates used by OpenKit.

## Template Files

- `config.yaml` -> `.openkit/memory/config.yaml`
- `derivation.yaml` -> `.openkit/memory/derivation.yaml`
- `queue.yaml` -> `.openkit/ops/queue.yaml`
- `rules/MEMORY_KERNEL.md` -> `.opencode/rules/MEMORY_KERNEL.md`

## Notes

- Legacy semantic-memory plugin files were removed.
- Active memory operations are CLI-based: `openkit memory init|doctor|capture|review`.
